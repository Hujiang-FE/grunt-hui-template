grunt-hui-template -- 模板引擎HUI.template的扩展插件. 
=============================

该项目包含了以下特性:

1. 可以将模板文件编译为模板js文件
2. 可以通过参数配置不同的输出选项

## APIs

### src
	
	指定模板文件的源

### dest
	
	指定模板文件的生成路径

### banner
	
	设置生成的模板js文件的banner，支持grunt.template模板

### footer

	设置生成的模板js文件的footer，支持grunt.template模板

### beautify

	设置是否要美化生成的js文件
